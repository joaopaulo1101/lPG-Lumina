# Lumina Clinic

Landing page premium para uma clínica estética fictícia, criada como case de portfólio front-end.

## Stack

- React
- Vite
- TailwindCSS
- Framer Motion

## Destaques

- Layout responsivo com estética premium e foco em conversão
- Componentização por layout, seções, UI e dados
- Animações de scroll reveal com suporte a `prefers-reduced-motion`
- FAQ acessível com `aria-expanded`, `aria-controls` e regiões nomeadas
- SEO base com meta tags, Open Graph, favicon e JSON-LD
- CTAs consistentes e navegação por âncoras

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Estrutura

```text
src/
  components/
    layout/
    sections/
    ui/
  data/
  lib/
  styles/
```

Projeto fictício desenvolvido para demonstrar UI engineering, UX, responsividade, acessibilidade e organização de código em uma landing page vendável.
